org.bdgenomics.adam.instrumentation

ADAMMetricsListener

class ADAMMetricsListener extends SparkListener

Spark listener that accumulates metrics in the passed-in ADAMMetrics object at stage completion time.

Note

This class relies on being run in the same process as the driver. However, this is the way that Spark seems to work.
